# SweetCupcake Management System

A Java-based desktop application developed using **Java Swing** and **Object-Oriented Programming (OOP)** principles for managing a cupcake shop. The system provides separate interfaces for **Managers** and **Cashiers**, allowing efficient product management, sales handling, and user authentication.

---

##  Overview

The SweetCupcake Management System is designed to simplify the daily operations of a cupcake shop. It features a graphical user interface (GUI) built with Java Swing and uses text files for data storage, making it suitable for academic projects and learning OOP concepts.

---

##  Features

###  Manager
- Login authentication
- Add new cupcake products
- Update existing products
- Delete products
- View available cupcakes
- Manage shop inventory

###  Cashier
- Secure login
- View cupcake menu
- Process customer orders
- Calculate total bill
- Record sales information

###  General
- Role-based login system
- File-based data storage
- Java Swing GUI
- Object-Oriented Programming implementation
- Easy-to-use interface

##  Getting Started

### Prerequisites

- Java JDK 8 or later
- NetBeans IDE (Recommended)

### Installation

1. Clone the repository

```bash
git clone https://github.com/shenalhq/Cupcake-Management-System.git
```

2. Open the project in NetBeans IDE.

3. Build the project.

4. Run the project by executing:

```
Main.java
```
